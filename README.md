# RedRoverPuzzle

This console application will print the following given this string: (id, name, email, type(id, name, customFields(c1, c2, c3)), externalId)

- id
- name
- email
- type
  - id
  - name
  - customFields
    - c1
    - c2
    - c3
- externalId
 
And also to this output:

- email
- externalId
- id
- name
- type
  - customFields
    - c1
    - c2
    - c3
  - id
  - name
